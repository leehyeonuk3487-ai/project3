# 군복무 청년 상해보험 — 리스크 층화·계리 추정 + 예산 최적화

지자체가 운영하는 **군복무 청년 상해보험**의 보장 설계·예산 책정을 돕기 위한
데이터 파이프라인이다. 공개 통계를 **상해/질병/모집단/위험요인**의 4개 트랙으로
결합해 보장항목별 발생률을 추정하고, 지역·연령·성별로 집단 리스크를 층화하며,
지자체별 현역 모집단을 자동 산출해 **예산을 최적화**한다.

> ⚠️ **포지셔닝(정직성 우선).** 개인의 사고를 예측하지 않는다. 조사 데이터
> (CHS/KNHANES)는 횡단면 자료라 개인 부상 라벨이 없어 개인 인과 추정이 불가능하다.
> 대신 공개 발생률(KDCA/KOSIS)과 위험요인 분포(CHS)를 결합한 **집단 단위 계리
> 추정 + 리스크 층화**를 제공한다. 층화 IRR은 **지역(지자체) 차원**에서 유효하며,
> 미션은 *"예산 효율화 → 같은 예산으로 더 많은 장병이 더 많은 혜택"* 이다.

## 데이터 트랙 (활용 데이터)

| 트랙 | 데이터 | 용도 |
|---|---|---|
| **상해** | KDCA 중증외상(발생·치명률·장해율·중증장해율), 퇴원손상(퇴원율·골절분율·치료결과·재원일수) | 상해사망·후유장해·골절·입원 발생률 |
| **질병** | KDCA 비외상성 중증손상(발생·치명률·장해율·중증장해율), KOSIS 사망원인(전체사망) | 질병사망·질병후유장해 발생률 + 전체사망 envelope 검증 |
| **모집단** | KOSIS 장래인구추계(시도×성×5세), 병무청 신체검사(등급→현역율, BMI 분포) | 지자체 현역 N 자동산출 + 미래 투영 |
| **위험요인** | CHS(시군구 행태·BMI), KNHANES(측정 BMI·흡연) | 생태회귀·층화 + 자가보고 보정 |
| **최적화** | 지자체 급부표 10곳(연구 보고서) | 기대청구액·권장 보험료·예산 배분 |

처리 데이터(CHS/KNHANES, utf-8-sig)와 raw(KDCA utf-8, KOSIS 사망원인 cp949)의
인코딩이 섞여 있어 `src/data/encoding.py`가 자동 판별한다.

병무청 OpenAPI는 `src/data/mma_api.py`로 키(.env)·캐시·폴백을 구현했으나, 본
실행 환경은 egress 허용목록이 `apis.data.go.kr`을 차단한다. 동일 신체검사 통계가
이미 `data/raw/mma_judgment_status/*.csv`로 커밋돼 있어 모집단 트랙은 그
스냅샷으로 동작한다(호스트 허용 시 클라이언트로 갱신 가능).

## 방법론

**발생률(비례 보정).** KDCA 발생률은 성·연령·시도가 각각 주변분포로만 제공된다
(joint cell 없음). 계리 표준인 비례 보정으로 결합한다:
`rate(sex,age,sido) = national × R_sex × R_age × R_sido`. 퇴원손상표는 성×연령
교차표라 그대로 쓰고 지역축만 중증외상 상대위험을 proxy로 적용한다.

**사망·장해 분해.** 상해/질병 사망 = 발생률 × 지역별 치명률(분율 검증: 사망건수
/발생건수). 후유장해는 정액에 **지급률 보정계수**(중증장해 비중 × 100% +
경증 × 0.3)를 곱해 과대추정을 교정한다. 상해사망은 협의(중증외상)와 광의(손상
퇴원율 × 치료결과 사망분율)를 함께 보고한다.

**모집단.** 현역 stock ≈ (20-24세 남성 인구/5) × 현역판정비율 × 평균 복무연수.
경기도 추정 9.8만명이 보고치(6.8만~10.5만)와 부합. 인구추계로 미래연도 투영.

**위험요인·층화.** CHS 지역 위험요인 ↔ KDCA 발생률 생태학적 quasi-Poisson 회귀
(과산포 보정). CHS 자가보고는 KNHANES 측정값으로 보정(비만 +10%). 실측 발생률로
저/중/고 3분위 층화 + IRR.

**검증.** 개인 라벨이 없어 AUC 대신 **LORO(leave-one-region-out) 캘리브레이션**과
**예측구간**을 전면에 둔다.

```bash
pip install -r requirements.txt
python -m scripts.run_slice1     # 보장항목 발생률(상해+질병 트랙)

```

주요 산출(실데이터):
- **상해+질병** 보장항목별 시도 발생률 — 지역 격차 최대 3.2배.
- **모집단 자동화** — 경기 현역 9.8만(보고치 부합), 전국 20대 남성 2024년 336만
  → 2030년 270만(인구절벽).
- **생태회귀** — 흡연 SD당 IRR 1.40(p<0.01); KNHANES 보정 반영.
- **층화 IRR 3.95** — 지역 차원 유효.
- **예산** — 경기 1인당 기대청구액 1.9만원(미매칭 항목 제외 하한), 후유장해
  지급률 보정, 예산 제약 시 사망>후유장해>질병>골절>입원 배분, 다년 투영.
- **검증** — LORO MAE 31%↓, 캘리브레이션 0.83, **예측구간 적중 94%**,
  사망 envelope·질병사망≤발생 정합성 통과.

## ML 보강 모듈 (M0–M3, 통계 백본에 추가)

검증된 통계 백본(생태 포아송 + LORO + 예측구간)은 유지하고, ML은 모듈로만 추가한다.
모든 신규 모델은 교차검증 + 정규화 + 베이스라인 대비 성능을 보고하며, 베이스라인을
못 이기면 채택하지 않고 그 사실을 기록한다(시드 고정·재현성).

- **M0 사망 사인 분리** (`data/mortality.py`) — 사인분류 사망표(22개 ICD-coded leaf,
  20대남·17시도·2021–24)를 질병사/상해사/**자살(면책)**/기타 4범주로 **결정적 ICD-10
  매핑**(추측 없음). MECE 검증: 4범주합 6,811 = 전체사망 6,812(오차 0.015%). 질병사망=
  disease 직접·상해사망=external(자살제외) 직접으로 **기존 proxy 폐기**, 자살(20대남
  사망의 51%)은 면책 분리(예산 미포함). 4년 풀링·인구분모 율 + Marshall 경험적 베이즈.
- **M1 모집단 연결** (`models/population.cohort_profile`) — 병무청 BMI 분포를 실제
  연결(이전 미호출 dead code 해소) + KNHANES 만성질환·행태 피처 적재(`load_knhanes_features`).
- **M2 개인 손상위험 ML** (`models/injury_ml.py`) — KNHANES 손상경험(ij_expr=AC1_yr
  정제, N=5,730·양성 6.7%)을 전 연령·전성별 풀링 학습(20대만 학습 금지)·age·sex 피처.
  L2 로지스틱·LightGBM, 계층 5-fold + 연도 홀드아웃, 불균형 가중. **정직 보고: AUC
  modest(~0.55–0.57 CV)로 age+sex 베이스라인을 홀드아웃에서 못 이김 → baseline 채택.**
  위험요인 OR/gain(BMI·음주·남성↑)은 집단 단위 참고만 — 개인 장병 스코어링 금지.
  ⚠️ 자가보고·일반인구 손상 → 군 훈련손상과 다름(전이 한계).
- **M3 셀 패널 GBM** (`models/panel.py`) — 성×연령×연도 손상 패널에서 LightGBM이
  비례보정(로그가산) 베이스라인을 **시간 홀드아웃 RMSE 0.337→0.219, k-fold
  0.247→0.133으로 이겨 채택**. ⚠️ 지역 joint 관측 부재로 공간 CV 불가 → 지역은
  비례보정 백본 유지.
- **모듈 A 셀 패널 발생률 GBM** (`models/cell_panel.py`) — M2의 개인예측 음성결과를
  보완하는 **집단(셀) 발생률 학습·검증**. M0의 시도×성×연령×연도 joint 사망으로 M3가
  불가했던 **leave-one-시도-out 공간 CV + 최근연도 시간 CV**를 LightGBM Poisson(노출가중
  rate)으로 수행, 비례보정·단순평균 베이스라인과 정직 비교. **채택 게이트=공간·시간 모두
  deviance↓ + 캘리브레이션 [0.5,2.0] 안정**. 결과:
  · **전체사인(2,890셀, 2020–24)**: 단순 구조모델(age×sex×year)로 충분 → GBM 미채택, baseline 유지.
  · **외인 상해사망 — 이전(136셀, 2021–24)**: 공간CV 신호 있으나 시간CV 캘리브레이션 7.5 폭주 → 미채택.
  · **외인 상해사망 — 확장(666셀, 2005–24)**: 주민등록 연앙인구(2005–19)+추계(2020–24) 결합 분모로
    패널 5배 확대 → **시간CV 캘리브레이션 7.5→0.76 안정(소스전환 2020 제외 시 0.79, 경계 불연속
    무영향), 공간·시간 모두 baseline 우세 → GBM 채택(✅ 첫 도메인 타깃)**. 단 승리 동인은 외인사망
    20대남 **70% 감소 추세(2005년 27.8→2024년 8.4/100k)**의 year 추세 외삽(gain 68%)이며, 횡단
    행태 신호는 **흡연(year 제외 시 45%)** 주도. ★검증본 GBM external surface(`external_gbm_surface`)는
    제공하되 **M0 직접 external 백본을 대체하지 않음**(대원칙: 백본 대체 금지). 신규 사망 데이터 없음.

- **M4 코호트 리스크 통합·보정** (`models/cohort.py`) — M1 코호트 프로파일(병무청 BMI +
  CHS 20대남 행태)에 LORO로 검증된 생태회귀 β(IRR)를 적용해 **시도별 코호트 리스크 지수**
  (전국 인구가중=1.0, 흡연·비만 주도; 경북 1.99·서울 0.57) + 요인기여를 산출. 현역 BMI≥25
  (과체중+비만 34.3%)는 일반 20대남(39.9%)보다 낮으나 군 훈련손상이 건강선택을 상쇄할 수
  있어 **손상률에 자동 미적용(민감도만)**. 통합 기대청구액은 검증 surface 기준(이중계산 방지),
  지수는 해석/민감도로 병기. 집단 단위·개인 스코어링 금지(ecological fallacy 주의).
- **M5 계리 보험료 산정** (`optimize/premium.py`) — budget의 단순 ×1.25를 **순보험료 +
  위험할증(복합포아송 표준편차원리 α·SD[S]/N) + 사업비 로딩**으로 고도화. 경기도 순 22,033 +
  위험할증 2,502(α=1) → 사업비 20% → 총 30,668원/인(보고 47,920의 −36%, 하한 일관). **소형
  풀(화천군 CV 0.30)은 대형(경기 0.11)보다 1인당 위험할증↑** — 신뢰도/풀링 효과를 계리적으로
  반영. α·사업비율 민감도표 제공(가정 노출).

- **M6 시나리오·민감도 + 복지가중 LP** (`optimize/budget.optimize_benefit_lp`,
  `optimize/scenarios.py`, `models/projection.py`) — ① **예산 제약하 복지가중 혜택 최대화**
  fractional-knapsack LP(scipy.linprog) + 생명·중대장해 완전보장 floor(고위험 배제 금지).
  ★목적함수를 '머릿수'가 아니라 '중대도 가중 혜택'으로 둔다 — 머릿수 최대화는 밀도=1/지급액이라
  *싼 보장 편향*으로 퇴화(미션 '더 많은 **혜택**'과 불일치). 가중치는 LP 결과서 역산하지 않고
  GBD 장애가중 류 **독립 근거로 사전 고정**(`DEFAULT_WELFARE_WEIGHTS`). 고정 우선순위 그리디와
  정직 비교(`compare_allocation`, 복지가중 기준). ★현 급부표는 floor 위 재량항목이 골절·입원
  2개뿐·지급액 거의 동일(knife-edge)이라 LP=그리디 **동률** — 이를 숨기지 않고, LP 필요성은
  (a)가중치 민감도(`weight_sensitivity`: 입원>골절 가중에서만 우위 → 역산 아님 투명 공개),
  (b)섭동 민감도(`perturbation_sensitivity`: 우선순위 가정만 흔들면 LP가 그리디 +23~46명 추월)로 입증.
  ② 민감도 토네이도(보장한도·사업비가 최대 변동요인) + 인구절벽(전국 20대남 336만→270만, −19.7%)×
  외인추세 다년 예산밴드. ③ **미래 외인추세는 parametric 로그선형 외삽**(GBM은 트리라 추세 외삽
  불가→공간검증 역할 유지) — 점추정 아닌 **밴드**(장기 −6.9%/년·둔화 −3.3%/년·평탄 0%), 구조변화
  미반영 한계 명시. **★현재 pricing은 M0 직접관측 백본 불변, 외삽은 미래 투영에만 사용**(MECE envelope 보존).
- **M7 통합 리포트** (`validation/integrated_report.py`) — 전 모듈을 라이브 호출해 평가기준별
  매핑(공공데이터·AI검증·독창성·발전가능성)·정직성/한계·윤리/미션·출처/라이선스를 하나의
  마크다운으로 종합. 생성: `python -m src.validation.integrated_report` → `reports/M7_integrated_report.md`.

## 한계·정직성

- KNHANES 손상 라벨(M2)은 자가보고·일반인구라 군 훈련손상과 다르고 예측력 modest →
  **개인 예측 불가**(개인 스코어링 금지), 집단 층화·위험요인 참고에 한정.
- 발생률 결합은 주변 독립 가정의 근사, 생태회귀는 17개 시도라 흡연만 robust.
- 예산은 핵심 보장항목 기준 **하한**(정신질환·수술비 등 미매칭 항목 제외).
- 시군구(화천군 등) 현역 N은 시도 추계라 과대 → 명시 입력 권장(광역은 자동).
- 사망 사인 분리(M0)는 22개 leaf의 ICD-10 결정적 매핑. 질병사망·상해사망은 사인별
  직접 사망률, 자살(X60–84)은 면책 분리(보장·예산 미포함), 전체사망은 envelope 검증.
- 미사용(의도적): 손상 기전·부위별(보장 세분화 확장), 병무청 신장·체중(BMI로
  대체)·역종·연감 PDF — 활용 CSV 18/24(75%).
